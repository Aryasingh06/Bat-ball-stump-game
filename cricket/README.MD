# Bat Ball Stump Game
An interactive web-based game built using HTML, CSS, and JavaScript where users can play against the computer by selecting Bat, Ball, or Stump.

## Features
- Interactive gameplay (Bat, Ball, Stump)
- Random computer move generation
- Live score tracking (Win / Loss / Tie)
- Score persistence using localStorage
- Enhanced UI with hover effects and smooth transitions

## Tech Stack

- HTML
- CSS
- JavaScript

## Game Logic

- The user selects an option (Bat, Ball, or Stump)
- The computer randomly generates a move
- The result is determined using conditional logic
  (Bat beats Ball
  Ball beats Stump
  Stump beats Bat
  Same choices result in a Tie)
- The score is updated and stored in localStorage.
